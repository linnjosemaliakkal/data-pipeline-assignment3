# Assignment 4 Reflection

## 1. What Worked Well
The integration of multiple datasets worked effectively.  
The pipeline structure from Assignment 3 made it easy to extend the project.  
The Streamlit app successfully presented the data and statistical results in an interactive format.

## 2. Challenges Faced
One challenge was aligning datasets using the correct date format.  
Another issue was ensuring that the holiday dataset matched the existing data range.  
Debugging errors related to missing columns also required careful attention.

## 3. Assumptions
- Holidays may influence market behavior  
- Sentiment reflects investor psychology  
- Data sources are accurate and reliable  

These assumptions may not always hold true in real-world scenarios.

## 4. Limitations
- Correlation does not imply causation  
- External factors such as global news are not included  
- The holiday dataset is simplified and may not represent real market conditions fully  

## 5. Improvements for Future Work
- Add more external datasets such as news sentiment or macroeconomic data  
- Improve feature engineering for deeper analysis  
- Enhance visualization with interactive filters  
- Use larger datasets for more robust conclusions  

## 6. Overall Learning
This assignment helped me understand how to:
- Extend a data pipeline  
- Apply statistical testing correctly  
- Build an interactive dashboard  
- Interpret results critically  

It demonstrated how raw data can be transformed into meaningful insights.
